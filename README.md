# skills-packs

[Spanish](./README.es.md)

A collection of AI agent skill packs for common development workflows, grouped by technology and use case. Built for developers who got tired of repeatedly running the same commands and want quick access to their essential skills.

## Motivation

Ever find yourself typing the same commands over and over? This project solves that by packaging your most-used skills into organized, installable units. No more repetitive setups—just install what you need and go.

## Installation

Install skills using the [Vercel Skills CLI](https://skills.sh):

```bash
npx skills add ivanTsxx/skills-packs/skills/next
```

This will prompt you to select which skills from the pack you want to install.
